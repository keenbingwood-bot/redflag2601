Vercel Analytics Test

Testing Steps:

1. Check if Analytics script is loaded

Open browser Developer Tools (F12) and check:

  • Network tab for va.vercel-scripts.com requests
  • Console for Analytics initialization messages
  • Elements tab for <script> tags with va-vercel

2. Verify Analytics Configuration

Check the following in your application:

  • package.json includes "@vercel/analytics": "^1.6.1"
  • app/layout.tsx imports Analytics from "@vercel/analytics/next"
  • <Analytics /> component is added to the layout
  • Build completes successfully

3. Production Deployment

After deployment to Vercel:

  • Visit your Vercel project dashboard
  • Go to Analytics tab
  • Check for incoming traffic data
  • Verify page views are being tracked

4. Expected Analytics Script

The Analytics script should look like:

<script defer src="https://va.vercel-scripts.com/v1/script.js"></script>

5. Testing Commands

Run these commands to verify:

# Check if package is installed
npm list @vercel/analytics

# Test build
npm run build

# Check layout.tsx
grep -n "Analytics" app/layout.tsx
